# Neovim Setup
a simple setup for NvChad that I use

## Installation
open your terminal

### delete previous
clear previous nvim configs with
```bash
rm -rf ~/.config/nvim/
rm -rf ~/.local/share/nvim
```

### now install
clone this repository and start nvim

```bash
git clone https://github.com/TheKhoshhal/NvChad.git ~/.config/nvim && nvim
```

### Enjoy it
